# POS System

A complete Point of Sale system built with Flask and PostgreSQL.

## Features
- User authentication with roles (Admin, Cashier, Inventory)
- Product management
- Inventory tracking
- Sales processing
- Receipt printing
- Customer management

## Deployment on Render

This app is configured for deployment on Render.com.

## Local Development

1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python web_server.py`
4. Visit: http://localhost:5000

## Default Users
- Admin: admin/admin123
- Cashier: cashier/cashier123
- Inventory: inventory/inventory123
